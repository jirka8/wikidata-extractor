\Wikidata API

- napojeni na wikidata api - skript bude umet stahovat a sestavovat sparql dotazy - jde nam konkretne o data mest a sidel v jednotlivych zemi
- na vstupu budeme definovat zemi a pozadovana data k jednotlivym sidlum - zakladni data budou item_id, nazev, gps souradnice, postal code, nuts code, lau code
- vystupem by mel csv soubor
- skript at je znovupouzitelny - parametry jako zeme, zakladni data a dalsi at jsou nastavitelne pres konfiguracni soubor - muze byt vice konfiguracnich souboru pro pozdejsi pouziti (strukturu a typ necham na tobe)
- technologicky stack - python - do budoucna moznost postavit nad tim uzivatelske rozhrani
